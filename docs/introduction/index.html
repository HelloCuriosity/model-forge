Introduction

Model Forge is a library that automates model generation for use in tests and currently supports the auto generation for the following types:

Types​

  • Boolean
  • Byte
  • Calendar
  • Char
  • Date
  • Double
  • Enums
  • File
  • Float
  • Int
  • Instant
  • Long
  • Short
  • String
  • UByte
  • UInt
  • ULong
  • UShort
  • UUID

Collections​

  • List
  • Map
  • Set
